======================================
SIBID
======================================
Biblioteca Digital del SERNANP

Concepto
--------
Es el repositorio de acceso público cuya finalidad es difundir la información y
conocimiento generados por las áreas naturales protegidas al público en general
interesado.

------------
SADT
------------
Sistema de Archivo Digital Técnico 

Estructura del sistema
----------------------
El archivo digital está diseñado como una estructura temática de carpetas con el fin de 
darle la máxima utilidad y accesibilidad, implantado en un entorno físico de servidor de
archivos, y que a través de un módulo web informático, el cual pertenece al Sistema
Integrado de Información, se brinda el acceso a diferentes usuarios para realizar la
búsqueda y registro, permitiendo así dar a conocer la producción documentaria
reportada por diferentes unidades y/o jefaturas. A ser un módulo web informático, se
puede acceder de cualquier lugar del país teniendo acceso a la red de Internet.
